 Brick City Bite: Things to Do in Newark, NJ

An interactive map built with [Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/) that highlights cultural venues, nightlife, museums, and parks in Newark, NJ.  
Users can filter locations by category, view details in a sidebar, and click markers to see popups with descriptions.
